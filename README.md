# おだんご重力砲台

マウスカーソルまたはタップした位置へ砲身を向け、チャージした串を発射する
2D物理パズルゲームのプロトタイプです。

## 起動

```powershell
npm install
npm run dev
```

## 操作

- マウス移動: カーソル位置へ照準
- プレイ領域をタップ: タップ位置へ照準を保持
- `Space`、マウス左ボタン、発射ボタン長押し: チャージ
- キーまたはボタンを離す: 現在の照準角度で発射
- `Esc`: ポーズ
- `R`: リトライ
- `D`: 予測軌道の表示切り替え

## 確認

```powershell
npm test
npm run build
```
